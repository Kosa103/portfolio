This program is written in JavaScript, Node.js and some practice code using React framework exists.

This is a basic webserver/smartHome app for Raspberry Pi. The code uses the RPi's GPIO pins to switch lights, measure air quality data and signal the opening of a door which triggers an e-mail for the owner. Socket.io is used to display this dynamically changing data on a very basic frontend webpage.
